# Internship Project

The **Demografy** Streamlit app and all application code live in the **`Demografy/`** directory (including the git submodule when cloned).

## Run the app

```bash
cd Demografy
python -m venv .venv && source .venv/bin/activate   # optional
pip install -r requirements.txt
streamlit run app.py
```

Full setup, environment variables, and architecture notes are in **`Demografy/README.md`**.
